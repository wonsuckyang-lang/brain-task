⚙️ 동기화 설정
📱 PC ↔ 휴대폰 자동 동기화
JSONBin.io 무료 서비스를 사용합니다.
아래 순서대로 1회만 설정하면 이후 자동으로 동기화됩니다.
  1. jsonbin.io 접속 → 무료 회원가입 (Sign Up)
  2. 로그인 후 우측 상단 프로필 → API Keys 클릭
  3. SECRET KEY 복사 → 아래 칸에 붙여넣기
  4. 빈(Bin) 자동 생성 버튼 클릭 → Bin ID 자동 입력됨
  5. 저장 후 폰에서도 동일한 앱 파일을 열고 같은 KEY·ID 입력
